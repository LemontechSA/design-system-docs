id: icon
category: layout
title: Icon
description:
  Icon is the most abstract component on top of which all other Chakra UI
  components are built. By default, it renders a `div` element
---

## Import

```js
import { Icon } from "lemon-system";
```

## Usage

```jsx
<Icon className="text-primary-01 p2 hover:bg-secondary-01">
  This is a Icon
</Icon>
```

### as prop

You can use the `as` prop to change the element render, just like
styled-components.

```jsx
<Icon as="button" className="p2 border rounded">
  Button
</Icon>
```
