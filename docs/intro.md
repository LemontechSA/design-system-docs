# Get started

## Installation

Instalar la librería en el proyecto:

```bash
npm install lemon-system
```

Implementar el ThemeProvider en el archivo root de la aplicación de React:

```jsx
// index.js

ReactDOM.render(
  <ThemeProvider>
    <App />
  </ThemeProvider>,
  document.getElementById("root")
);
```
